# 📚 LMS Full Stack Project (MERN)

A **Learning Management System (LMS)** built with the **MERN stack** to showcase full‑stack development skills.  
This project demonstrates **authentication, role‑based access, course management, and responsive UI**.

## Live Link: https://lms-by-ajitesh.netlify.app/

## Demo Link: https://youtu.be/U-LuV3jjXbw?si=b2VHdniiEM-tT8AZ

---

## 🚀 Tech Stack
- **Frontend:** React + Vite, TailwindCSS / shadcn/ui
- **Backend:** Node.js + Express
- **Database:** MongoDB (Mongoose ODM)
- **Authentication:** JWT + bcrypt
- **Other:** REST APIs, CORS, dotenv, Formik + Yup for validation, Redux

---

## ⚙️ Features
- 🔐 User Authentication (Register, Login, Logout)
- 👩‍🏫 Role‑based access (Student, Instructor, Admin)
- 📖 Course creation, enrollment, and management
- 🛡️ Protected routes with JWT middleware
- 🎨 Responsive UI with modern design
- 🔔 Toastify notifications for success/error feedback

## 🛠️ Getting Started

### Step 1: Fork the Repository
Click the **Fork** button on GitHub to create your own copy.

### Step 2: Clone the Repository
```bash
git clone https://github.com/GowlikarAjitesh/LMS_fullStack
```

### Step 3: Install the dependencies and start the project
Frontend:
```bash
cd frontend
npm install
npm run dev
```

Backend:
```bash
cd backend
npm install
npm run dev
```
